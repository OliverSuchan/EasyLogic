Ausführliche Beschreibung

GUI-Klasse, welche den gesamten logischen
Programmablauf visualierst und bedienbar macht.
Wie zum Beispiel:

  • Elemente speichern/ laden
  • Map bearbeiten
  • Simulation starten/ stoppen
  • Nächster/ vorheriger Simulationsschritt
  • Zoom
